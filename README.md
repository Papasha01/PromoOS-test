Реализован API для управления задачами на .NET 9 (Minimal API).

**Функционал:**
*   **Создание задачи:** POST-запрос сохраняет новую задачу с валидацией названия.
    ![Создание задачи](https://github.com/user-attachments/assets/bfbe2b45-4cb4-4173-9305-631bb3a93ac2)
*   **Получение списка:** GET-запрос возвращает все существующие задачи.
    ![Список задач](https://github.com/user-attachments/assets/ffa80003-d9dd-4b3c-8d46-b361c61ac50e)

**Технические детали:**
*   **БД:** PostgreSQL + EF Core. Данные корректно сохраняются и читаются.
    ![Записи в БД](https://github.com/user-attachments/assets/2967037a-e8aa-4ee0-bbbe-b398f65c2f4f)
*   **Конкурентность:** Обработка одновременных запросов на завершение одной задачи через оптимистичную блокировку (возврат 409 Conflict).
*   **Интеграция:** При успешном завершении задачи отправляется событие в RabbitMQ. Реализована устойчивая работа: если брокер недоступен, ошибка логируется, но транзакция в БД сохраняется.
    ![RabbitMQ](https://github.com/user-attachments/assets/4a769cfa-b1bf-42fc-a848-5535367957f1)
*   **Shutdown:** Корректное освобождение ресурсов RabbitMQ при остановке приложения.
